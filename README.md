# TOTEN-Burguer-Tech
# 🍔 BURGER TECH

Sistema de totem digital para pedidos de hamburguer desenvolvido com **HTML, CSS e JavaScript puro**.  
O projeto simula o funcionamento de um autoatendimento de lanchonete, permitindo que o cliente:

- informe seu nome;
- escolha um lanche;
- adicione combo;
- visualize o resumo final do pedido;
- reinicie o sistema para um novo atendimento.

---

## 📸 Visão Geral do Projeto

O sistema possui 4 etapas principais:

1. **Tela de Login**
   - Cliente digita o nome.

2. **Tela de Cardápio**
   - Escolha do hambúrguer.

3. **Tela de Combo**
   - Cliente escolhe se deseja adicionar batata + refrigerante.

4. **Tela Final**
   - Exibição do pedido finalizado com total da compra.
